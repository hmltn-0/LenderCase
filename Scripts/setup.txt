Setup Instructions:

1. Create an account at the Language Model API provider's website.
2. Enable the API access and obtain an API token.
3. Replace 'YOUR_API_TOKEN_HERE' in the text_extraction_script.py with your actual API token.
4. Ensure you have the required Python package installed by running:
   pip install requests
5. Specify the path to the PDF file in the script and run it using:
   python Scripts/text_extraction_script.py
